## 課程內容

2.1.硬體架構與分類  
* MCU 主要分類方式  
* 全球通用型 MCU 主要供應商  
* 台灣通用型 MCU 主要供應商  
* arm Cortex-M 以STM32為例  

2.2. arm Cortex-M演進  
* 常見 Arm 晶片CPU等級及指令集  
* arm Cortex-M 指令比較表

2.3. 基本週邊控制  
* RP2040 資料手冊  
* Pi Pico 開發板及 RP2040 接腳圖  
* 工作電壓及電壓轉換 (Level Shift)  
* RP2040 方塊圖  
* RP2040 匯流排及週邊  
* RP2040 位址配置  
* 系統啟動及工作模式  
* 工作（睡眠）模式  
* 工作時脈 (Clock)   
* 計時器 (Timer) 與實時時鐘(RTC)  
* 基本輸出入 (GPIO) ─ 數位輸出入 (DIO)  
* 基本輸出入 (GPIO) ─ 類比輸入 (ADC)  
* 基本輸出入 (GPIO) ─ 類比輸出 (PWM)  
* 串列通訊 (UART / COM)  
* 匯流排通訊 (IIC / I2C / I2C)  
* 串列週邊介面 (SPI)  
* 直接記憶體存取 (DMA)  
* 看門狗 (Watchdog)  
* USB & Virtual COM  

2.4. 即時作業系統  
* MCU 即時作業系統  
* FreeRTOS  
* Mbed OS  
* Zephyr
